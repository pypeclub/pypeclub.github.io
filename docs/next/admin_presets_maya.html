Maya instance scene types

It is possible to set when to use .ma or .mb for:

  • camera
  • setdress
  • layout
  • model
  • rig
  • yetiRig

Just put ext_mapping.json into presets/maya. Inside is simple mapping:

{
  "rig": "mb",
  "camera": "mb"
}

Note that default type is ma
